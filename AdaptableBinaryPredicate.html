Adaptable Binary Predicate

Category: functors Component type: concept

Description

An Adaptable Binary Predicate is a Binary Predicate that is also an Adaptable Binary Function. That is, it is a Binary Function whose return type is bool, and that includes nested typedefs that define its argument types and return type.

Predicate, Adaptable Binary Function
